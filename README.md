# 🤖 SupportGPT - AI Customer Support Agent

SupportGPT is an AI-powered customer support assistant built using Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and Agentic AI.

## 🚀 Tech Stack

- Python
- Ollama
- LangChain
- LangGraph
- ChromaDB
- Streamlit
- HuggingFace Embeddings
